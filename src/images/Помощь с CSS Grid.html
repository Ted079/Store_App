1. Изменение высоты для определённых элементов

Для элементов с высотой 60px, добавь классы, такие как short-card, и в CSS укажи нужную высоту.

2. Изменение ширины для элементов

Изменения в CSS:

scss
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
  gap: 0.5rem;
  margin-top: 2rem;

  .card {
    background-color: var(--generic);
    padding: 1rem;
    border-radius: 10px;
    display: flex;
    align-items: center;
    justify-content: center;
    flex-direction: column;
    gap: 0.5rem;
    font-size: 14px;
    color: var(--grey);
    text-align: center;
    transition: background-color 0.3s ease;

    &:hover {
      background-color: var(--dark-sea);
    }

    .icon {
      width: 32px;
      height: 32px;
      fill: var(--grey);
      stroke: var(--grey);
    }
  }

  .shortCard {
    height: 60px;
  }

  .wideCard {
    width: 100px;
  }
}

Что здесь происходит:

  • Для карточек с высотой 60px добавлен класс .shortCard, который задаёт нужную высоту.

  • Для карточек с шириной 100px добавлен класс .wideCard, который устанавливает фиксированную ширину.

  • При этом остальные карточки будут следовать стандартному стилю.

Теперь ты можешь контролировать высоту и ширину карточек, комбинируя классы shortCard и wideCard в зависимости от нужд.
